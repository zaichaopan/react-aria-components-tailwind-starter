# Button
 
A <a href="https://react-spectrum.adobe.com/react-aria/Button.html" target="_blank">**button**</a> allows a user to perform an action, with mouse, touch, and keyboard interactions.

<Canvas of={ButtonStories.BasicExample} />

## Button colors

Use the `color` prop to set the button color. The default color is `accent`. Available colors are `accent`, `red`, and `green`.

<Canvas of={ButtonStories.ButtonColors} />


### Custom background and color

- Use `[--btn-bg:*]`. <a href="https://til.jakelazaroff.com/css/swap-between-black-and-white-text-based-on-background-color" target="_blank">**Text and icon color will be auto swapped based on the background**</a>.

- Use the `bg-*` and `text-*` utility.

<Canvas of={ButtonStories.CustomBackgroundAndColor} />

## Button sizes

Use the `size` prop to set the button size.

<Canvas of={ButtonStories.ButtonSizes} />

## Button variants

Use the `variant` prop to set the button variant. The default variant is `solid`. Available variants are `solid`, `outline`, `plain`, `link` and `unstyle`.

<Canvas of={ButtonStories.ButtonVariants} />

## Button with icons

Use the <a href="./?path=/docs/icon--docs" target="_blank">**Icon**</a> component to place a decorative icon at the start or end of a button. The icon size is automatically scaled based on the button size.

<Canvas of={ButtonStories.ButtonWithIcons} />

## Icon only

Use the `isIconOnly` prop to render an `icon-only` button. For non-decorative buttons, use the `aria-label` on the <a href="./?path=/docs/icon--docs" target="_blank">**Icon**</a> component. The icon size is automatically scaled based on the button size.

<Canvas of={ButtonStories.IconOnlyButtons} />
